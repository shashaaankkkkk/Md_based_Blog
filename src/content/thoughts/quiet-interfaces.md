# Quiet interface design

I’ve been thinking a lot about software that doesn't demand attention. Most modern apps use notification badges, animations, and saturated colors to keep you engaged. 

What if software felt more like a physical object? A well-crafted notebook, a wooden desk, a smooth pen.

```js
function quiet(mind) {
  return mind.filter(thought => thought.isEssential);
}
```

This platform should be an example of that. Monospace fonts, off-white backgrounds, and lots of breathing room.
